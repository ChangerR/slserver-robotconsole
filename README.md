###slserver-robotconsole
---
- 运行此程序首先应安装 [nodejs](https://nodejs.org/download/release/latest-v0.12.x/node-v0.12.7-x86.msi)
- 安装此程序所使用的必要的库
```shell
#在此程序的根目录下运行
npm install
```
- 启动测试服务器
```shell
node server.js
```
- 启动网页服务器
```shell
node app.js
```
- 打开URL http://localhost:9000/

>在服务器控制台中输入`7:::hdgd=100,roll=1,pitch=1,yaw=1,depth=1,temp=1`可以改变显示状态
